Documentation

Welcome to the Advanced Intrusion Detection System (IDS) documentation. This page provides an overview of the datasets, model, and features of the application.

Datasets

  • Cybersecurity Intrusion Data: Contains session-level data with features like network packet size, protocol type, and attack detection labels.
  • Augmented Dataset: Includes detailed network flow metrics for intrusion detection, such as flow duration, packet lengths, and labels like DDoS and PortScan.

Model

The application uses a deep learning model trained on the augmented dataset. The model is capable of detecting various types of network intrusions, including:

  • Denial of Service (DoS) and Distributed Denial of Service (DDoS) attacks
  • Port and vulnerability scans
  • Man-in-the-middle attacks
  • Malware communication

Features

  • Data Upload: Upload network traffic data in CSV or PCAP format for analysis.
  • Visualization: Generate bar and pie charts to explore dataset distributions.
  • Real-Time Monitoring: Continuously analyze network traffic for immediate threat detection.
  • Threat Detection: Identify and classify network intrusions with high accuracy.

How to Use

  1. Navigate to the Analyze page to upload your dataset.
  2. View the analysis results, including detected threats and confidence scores.
  3. Explore the Visualize page to generate charts for dataset insights.
  4. Refer to this documentation for detailed information about the app.
